# Product Requirements Document (PRD)

# Cortiq

Version: 0.1 MVP

---

# Product Vision

Cortiq is an AI-powered desktop platform that helps companies understand how work actually happens.

Instead of monitoring employees, Cortiq analyzes workflows, identifies productivity patterns, and provides intelligent recommendations to improve efficiency.

---

# Problem

Modern companies often don't know:

- where time is actually spent;
- why projects become delayed;
- which workflows are inefficient;
- where employees lose focus;
- which repetitive tasks should be automated.

Most existing software focuses on time tracking instead of workflow intelligence.

---

# Solution

Cortiq collects desktop activity (with user permission), transforms it into meaningful analytics, and uses AI to generate recommendations for individuals and teams.

The goal is to improve workflows, not monitor people.

# MVP Goals

The first version should allow users to:

- Login
- Start tracking
- Stop tracking
- Detect active applications
- Display dashboard
